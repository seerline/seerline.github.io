安装

ClChart可以与ES6模块，普通的JavaScript和模块加载器一起使用。

安装 ClChart

你可以通过以下几种方式安装 ClChart

npm

如果您使用模块加载器则可以直接在使用到clchart的地方引入文件

或者

html文件引入

在 ClChart 的 GitHub 上下载最新的 release 版本的clchart.js文件，然后将其引入到您的网页中
